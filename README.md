# twitter-clone-api

## Usage

```sh
$ git clone https://github.com/NasSilverBullet/twitter-clone-api.git
$ cd twitter-clone-api
$ docker-compose up
$ docker-compose run --rm db db/cli up
$ docker-compose run --rm db db/cli seed

# Please access to localhost:8080/users
```
